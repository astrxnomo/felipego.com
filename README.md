# Felipe Giraldo's Portfolio

Welcome to my personal portfolio! It showcases my work as a developer and the technologies I use. Here's what this project is built with:

- **Astro**: A modern front-end framework that allows for fast, optimized websites.
- **Tailwind CSS**: A utility-first CSS framework that allows for highly customizable user interfaces.
- **lucide-astro**: This is an icon library designed specifically for Astro.
- **Notion API**: Used to integrate and manage content from Notion within the website.
- **GitHub Actions**: This is a CI/CD tool that automates the deployment of the website in Vercel.


Feel free to explore the repository to understand how these technologies are used in the project.
